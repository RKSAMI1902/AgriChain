# AgriChain
1. ABSTRACT

Agriculture supply chains in India often suffer from lack of transparency, food fraud, and data manipulation. Traditional database systems allow modification of records, making it difficult to ensure authenticity.

This project proposes AgriChain, a blockchain-based web application that tracks agricultural products from farmer to retailer. Each transaction is stored as a block containing crop details, ownership information, and location. Each block is connected using cryptographic hashing, ensuring data immutability.

The system allows users to add crop records, transfer ownership, view the blockchain, and validate chain integrity. Any attempt to modify data results in validation failure. This project demonstrates how blockchain can enhance transparency and trust in agricultural supply chains.

OBJECTIVES

The main objectives of this project are:

To develop a blockchain simulation for agricultural tracking

To ensure data immutability using hashing

To provide transparent ownership transfer

To validate the integrity of stored data
